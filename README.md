# minecraft-interactive-dashboard
An interactive D3.js visualization pipeline for exploring and comparing Minecraft mob attributes from a structured CSV dataset.

Libraries:

D3.js: CSV data loading, hierarchical grouping, tree layout generation, scatter plotting, and interactive tooltips
JavaScript: Data preprocessing, selection logic, and event-driven interactions
HTML: Page structure and visualization containers
CSS: Styling, custom font integration, responsive layout tuning, and animation effects
